## Deployment notes

1. Remove `.env` from the repo (it's already removed in this build). Add Vercel environment variables:

- `VITE_SUPABASE_URL` — your Supabase URL (e.g. `https://<project>.supabase.co`)
- `VITE_SUPABASE_ANON_KEY` — your Supabase anon key

2. Ensure Vercel Project settings: Build Command `npm run build`, Output Directory `dist`, Node version 18.x.

3. Push to GitHub and Redeploy on Vercel.
